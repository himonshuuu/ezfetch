# ezfetch

A simple Python package for easy and efficient data fetching.

## Installation

```bash
pip install ezfetch
```

## Usage

```python
import ezfetch

# Example usage
data = ezfetch.get('https://api.example.com/data')
print(data)
```

## Features

- Simple API for HTTP requests
- Supports GET, POST, PUT, DELETE
- Handles JSON responses
